# meme-generator-cli
Meme Generator written in .NET

## Arguments
```
Input file
--image <filepath>
-i
--image "./image.jpg"

Output file
--out <filepath>
-o
--output "./output_image.jpg"

Text on the top of image
--top <string>
-t
--top "Hello"

Text on the bottom of image
--bottom <string>
-b
--bottom "to the real world"

Size of the font (in px)
--size <number>
-s
--size 24

Color of the font (in hex)
--color <hex color string>
-c
--color "#ffffff"


```
